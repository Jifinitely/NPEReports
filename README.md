# NP E-Report

Professional electrical testing and reporting app for Australian electricians, contractors, and field technicians.

NP E-Report is designed to simplify electrical documentation by allowing users to record test results, manage project reports, and generate professional PDF reports directly from iPhone and iPad.

---

# Features

- Electrical test reporting
- Circuit schedules
- RCD test records
- Fault loop impedance records
- Insulation resistance records
- PDF preview and export
- Project-based report management
- Mobile-first field workflow
- Offline-friendly app structure

---

# Intended Use

NP E-Report is built for:

- electricians
- electrical contractors
- service technicians
- commissioning teams
- project supervisors
- maintenance workers

This app is intended as a field documentation and reporting tool.

# Installation and Testing

## 1. Install Xcode

On your Mac:

1. Open the Mac App Store
2. Search for Xcode
3. Download and install Xcode
4. Open Xcode once after installation
5. Accept the licence agreement
6. Allow Xcode to install any required components

---

## 2. Download the App Project

From GitHub:

1. Open this repository
2. Click the green Code button
3. Select Download ZIP
4. Unzip the downloaded folder
5. Open the project folder

---

## 3. Open in Xcode

Inside the project folder, open:

NP E-Report.xcodeproj

If the project name is different, open the .xcodeproj file included in the folder.

---

## 4. Run in iPhone Simulator

In Xcode:

1. Select an iPhone simulator from the top toolbar  
   Example: iPhone 16 Pro or iPhone 17
2. Press the Run button ▶
3. Wait for Xcode to build the app
4. The iPhone Simulator will open automatically
5. NP E-Report will launch in the simulator

---

## 5. Run on a Physical iPhone

To test on your own iPhone:

1. Connect your iPhone to your Mac with a cable
2. Open the project in Xcode
3. Select your iPhone from the device list
4. Sign in with your Apple ID in Xcode if required
5. Enable Developer Mode on the iPhone if prompted
6. Press Run ▶

A free Apple Developer account can run the app locally for testing. A paid Apple Developer account is required for TestFlight or App Store distribution.
